# Hill Cipher — Encryption & Decryption 🔐

A web-based application that implements the Hill Cipher algorithm for secure encryption and decryption of text using matrix operations.

## 🚀 Features
- Encrypt plaintext using Hill Cipher
- Decrypt ciphertext back to original message
- Supports customizable matrix sizes (e.g., 2x2)
- Automatic key generation
- Displays computation details step-by-step
- Clean and user-friendly UI

## 🧠 Concepts Used
- Matrix multiplication
- Modular arithmetic (mod 26)
- Cryptography fundamentals
- Linear algebra in encryption

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
